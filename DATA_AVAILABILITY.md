# Data and Code Availability

This repository provides the processed feature matrices and source code needed to reproduce the main tabular analyses reported in the manuscript.

The BorodaTM benchmark table used in the analyses is provided as `data.xlsx`. The original benchmark should be cited through the BorodaTM publication.

The released processed matrices are sufficient to reproduce the main tabular experiments reported in the manuscript:

- `experiment/features/esm3_79/X_esm3_79.npy`
- `experiment/features/X_uniprot.npy`
- `data.xlsx`

Recorded result tables, generated figures, LaTeX files, and large intermediate extraction caches are intentionally omitted. Running the scripts will regenerate result files locally from the released data and feature matrices.
